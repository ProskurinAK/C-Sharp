В проекте реализован ансамблевый метод случайного леса на основании моделей обучающего дерева.
Проект состоит из 3 классов.
1 - class TrainingSample
В данном классе реализовано создание обучающих подвыборок для подачи их на простые модели деревьев.
Выборки создаются на основании методов Бутстрэпа и случайных подпространств.
Для каждой модели дерева будет своя уникальная подвыборка на которой оно будет обучаться.
Результат работы класса: Запись в файлы уникальных обучающих подвыборок для подачи на модели.
2 - class DecisionTree
Это класс обучения одного дерева из ансамбля. В качестве эвристики для обработки количественных признаков в дереве решений
было выбрано следующее: количественный признак сортируется по возрастанию, и в дереве проверяются только те пороги,
при которых целевой признак меняет значение.
Для формирования предикатов, в качестве критерия информативности, на основе которого осуществляется разбиение выборки
на каждом шаге, используется индекс Джини. Деревья строятся до тех пор, пока не иссякнет обучающая выборка.
Результат работы класса: Запись в файлы всех предикатов для каждого из деревьев
3 - class RandomForestClassifier
В данном классе происходят вызовы двух других классов, а так же реализованы функции предсказания результатов на основе
полученных предикатов и подсчёт количества совпадений и общий процент правильных предсказаний для каждой из моделей
