При использовании данного метода динамического изменения размера элементов управления при измененеии размеров формы, в том случае, когда элементы управления
распологаются напрямую на форме, происходит неккоректное изменение размера. В качестве решения проблемы можно располагать на фоме элемент управления Panel и 
добавлять другие элементы управления на Panel(как это происходит в данном решении), но в таком случае к элементам управления неполучается применить функционал
запрета изменения размера для выборочных ЭУ путём изменения их поля AutoSizeMode с GrowOnly на GrowAndShrink.
